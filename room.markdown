# 1.广州菱致计算机科技有限公司房间模块设计方案

##1.1 公司设置
>公司使用odoo自身的res_partner设置的数据表，取其中的公司

>公司设置用于应对一个集团下多个分店的业务场景,所有的设置表和业务表，都要包含公司id，

>公司数据使用 res.partner 下的 is_company = 1 的数据

>
